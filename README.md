# 🎣 Arundo — Онлайн довідник для рибалок України

Повноцінний вебдодаток для рибалок-любителів та спортсменів України. Платформа об'єднує інтерактивну карту водойм, енциклопедію риб з фільтрацією, особистий щоденник уловів та профіль користувача. Реалізована JWT-авторизація з рольовим доступом, завантаження зображень через Cloudinary, адміністративна панель та мультимовний інтерфейс.

## 📦 Локальний запуск

### 1. Клонувати репозиторій

```bash
git clone https://github.com/Ts4ndrii/Arundo.git
cd Arundo
git checkout develop
```

### 2. Встановити залежності (Frontend)

```bash
npm install
```

### 3. Налаштувати змінні оточення

Створіть файл `.env.local` у корені проєкту:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000
```

Для бекенду створіть `backend/.env`:

```env
MONGODB_URI=mongodb+srv://<username>:<password>@cluster.mongodb.net/arundo
JWT_SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
PORT=5000
```

### 4. Запустити бекенд

```bash
cd backend
npm install
node server.js
```

### 5. Запустити фронтенд

```bash
# У корені проєкту
npm run dev
```

Відкрити в браузері: http://localhost:3000

## 🔗 REST API — Основні ендпоінти

### 🔐 Авторизація

| Метод | Endpoint | Опис |
|-------|----------|------|
| `POST` | `/api/auth/register` | Реєстрація нового користувача |
| `POST` | `/api/auth/login` | Вхід, отримати JWT-токен |
| `GET` | `/api/auth/me` | Дані поточного авторизованого користувача |

### 👤 Профіль користувача

| Метод | Endpoint | Права | Опис |
|-------|----------|-------|------|
| `PUT` | `/api/users/profile` | авторизований | Оновити ім'я |
| `POST` | `/api/users/avatar` | авторизований | Завантажити аватар |
| `GET` | `/api/users/favorites` | авторизований | Список улюблених водойм |
| `PUT` | `/api/users/favorites` | авторизований | Оновити список улюблених |
| `POST` | `/api/users/favorites/:waterId` | авторизований | Додати водойму до улюблених |
| `DELETE` | `/api/users/favorites/:waterId` | авторизований | Видалити з улюблених |

### 🗺️ Водойми

| Метод | Endpoint | Права | Опис |
|-------|----------|-------|------|
| `GET` | `/api/water` | всі | Список водойм (з фільтрами) |
| `GET` | `/api/water/nearby` | всі | Водойми поруч (за координатами) |
| `GET` | `/api/water/:id` | всі | Деталі водойми |
| `POST` | `/api/water` | admin | Додати водойму (з фото) |
| `PUT` | `/api/water/:id` | admin | Оновити водойму |
| `DELETE` | `/api/water/:id` | admin | Видалити водойму |

### 🐟 Риби

| Метод | Endpoint | Права | Опис |
|-------|----------|-------|------|
| `GET` | `/api/fish` | всі | Список риб |
| `GET` | `/api/fish/:id` | всі | Деталі виду риби |
| `POST` | `/api/fish` | admin | Додати рибу (з фото) |
| `PUT` | `/api/fish/:id` | admin | Оновити рибу |
| `DELETE` | `/api/fish/:id` | admin | Видалити рибу |

### 🎣 Щоденник уловів

| Метод | Endpoint | Права | Опис |
|-------|----------|-------|------|
| `GET` | `/api/catches` | авторизований | Мої улови (admin бачить усі) |
| `GET` | `/api/catches/my` | авторизований | Тільки свої улови |
| `GET` | `/api/catches/stats/summary` | авторизований | Статистика уловів |
| `GET` | `/api/catches/:id` | авторизований | Деталі запису |
| `POST` | `/api/catches` | авторизований | Додати запис улову (з фото) |
| `PUT` | `/api/catches/:id` | авторизований / admin | Оновити запис |
| `DELETE` | `/api/catches/:id` | авторизований / admin | Видалити запис |

### ⚙️ Фільтри та статистика

| Метод | Endpoint | Опис |
|-------|----------|------|
| `GET` | `/api/filters/water-types` | Типи водойм для фільтра |
| `GET` | `/api/filters/fish` | Риби для фільтра |
| `GET` | `/api/stats` | Загальна статистика платформи |
| `GET` | `/api/health` | Стан сервера та БД |

## 🗃️ Колекції MongoDB

| Колекція | Призначення |
|----------|-------------|
| `users` | Користувачі (ролі: `admin`, `user`), аватари, улюблені водойми |
| `waterbodies` | Водойми (GeoJSON координати, фото, види риб, сезони) |
| `fish` | Риби (опис, наукова назва, фото, вага, довжина) |
| `catches` | Записи уловів (прив'язані до водойми та користувача) |
| `watertypes` | Типи водойм для фільтрації |
| `filterfish` | Риби для фільтрації на карті |

## 🔐 Авторизація та ролі

Перший зареєстрований користувач автоматично отримує роль **admin**. Усі наступні — роль **user**.

| Роль | Можливості |
|------|-----------|
| `admin` | Повний доступ: CRUD водойм, риб, типів водойм, перегляд усіх уловів |
| `user` | Перегляд карти та енциклопедії, ведення щоденника уловів, улюблені водойми |
| Гість | Перегляд карти та енциклопедії без авторизації |
